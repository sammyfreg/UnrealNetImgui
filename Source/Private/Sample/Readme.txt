===============================================================================
  NetImguiDemoActor (.h/.cpp)
===============================================================================
Example of using 'NetImgui' with 'Dear ImGui' inside an Actor class. 
The 'Dear ImGui' draws can be done from anywhere in the engine (in GameThread), not limited to 'AActor::Tick()' or an Actor class.
